\begin{abstract}
    \textit{Această lucrare abordează problema monitorizării sistemelor distribuite, comparând limitările arhitecturilor centralizate cu robustețea algoritmilor decentralizați. Proiectul propune implementarea unui protocol de tip \textit{Gossip} (Push-Pull Dynamic Average Consensus) pentru calculul mediei globale a resurselor (CPU) într-un cluster, fără un server central.}
    
    \textit{Analiza teoretică și practică evidențiază avantajele majore ale soluției propuse: eliminarea punctului unic de eșec (SPOF), scalabilitate masivă și toleranță la erori. Totodată, sunt discutate provocările specifice consensului dinamic, inclusiv gestionarea condițiilor de cursă (\textit{Race Conditions}) și impactul ieșirii nodurilor din rețea asupra acurateței datelor.}
\end{abstract}
